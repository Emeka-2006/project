Position Absolute: WHEN POSITIVE; TAKES THE ELEMENT TO THE DIRECTION YOU WANT THEN STARTS CREATING SPACE BETWEEN THE DIRECTION FROM THERE, LEFT: 10PX; TAKES THE ELEMET TO THE LEFT AND START MAKING SPACES IN THE LEFT. WHEN NEGATIVE; TAKES THE ELEMENT TO THE DIRECTION YOU WANT THEN STARTS MAKING THE ELEMENT MOVE OUT OF THE CONTAINER, LEFT: -10PX; TAKES THE ELEMENT TO THE LEFT AND START MAKING THE ELEMENT MOVE OUT OF THE CONTAINTER. Position Fixed: KEEPS THE ELEMENTFIXED IN PLACE EVEN WHEN YOU SCROLL, ALLOWING IT TO STILL BE VISIBLE. position
